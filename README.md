# SANE Portfolio

라이브 프로덕션, 데이터 정산, 웹사이트 제작, 콘텐츠 운영 및 비즈니스 역량을 소개하는 정적 포트폴리오입니다.

## 주요 페이지

- `index.html` — 홈과 핵심 업무 소개
- `about.html` — 세부 역량
- `works.html` — 대표 프로젝트와 협업 채널
- `contact.html` — 프로젝트 문의

## 배포

Vercel에서 별도 빌드 명령 없이 정적 사이트로 배포합니다.

## 운영 메모

- 문의 이메일은 `contact.html`의 `mailto:` 주소에 연결합니다.
- 이미지와 영상은 `assets` 폴더에서 관리합니다.
- 공통 디자인은 `css/style.css`, 메뉴와 화면 전환 효과는 `js/main.js`에서 관리합니다.
