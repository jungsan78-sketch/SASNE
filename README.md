# 정산 포트폴리오 템플릿 (정적/Cloudflare Pages)

## 파일 구조
- index.html (홈)
- works.html (포트폴리오 허브)
- works-daily.html / works-full.html / works-etc.html (카테고리)
- about.html (소개/업무 범위)
- contact.html (문의)

## 배포
GitHub에 업로드 → Cloudflare Pages 연결 → 빌드 커맨드 없이 배포.

## 커스터마이징
- 연락처/소셜: contact.html 수정
- 작업물 카드: works-*.html 카드 링크/썸네일/설명 교체
- 색/폰트: css/style.css에서 변수 변경
